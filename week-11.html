Learning Bayesian Networks and Graphical Models

Parameter learning estimates CPT values from data. Structure learning discovers network topology.

Parameter Learning with Complete Data

Maximum Likelihood Estimation (MLE): θ = count(value) / count(parent values). Simple; assumes IID data.

Incomplete Data and EM Algorithm

E-Step
Compute expected counts using current parameters.
M-Step
Update parameters using expected counts.

Structure Learning

  • Score-based: BIC/MDL score trade-off fit vs. complexity; search over DAGs
  • Constraint-based: CI tests identify independence; build structure from constraints

Neural Networks: Perceptrons and MLPs

Perceptrons single linear units; Multi-layer perceptrons (MLPs) composition via nonlinearities.

Perceptron Learning

w ← w + α(y - h(x))x where h(x) = 1 if w·x > 0 else 0
Converges for linearly separable; fails otherwise.

Neural Network Architecture

  • Input layer: Feature vectors
  • Hidden layers: Learned representations; nonlinear activations (ReLU, sigmoid, tanh)
  • Output layer: Predictions; softmax for classification, linear for regression
Universal Approximation

MLPs with hidden layer can approximate any continuous function (given enough neurons).

Backpropagation and Training

Backpropagation efficiently computes gradients via chain rule. Enables gradient descent on deep networks.

Algorithm

1. Forward pass: compute activations
2. Compute loss
3. Backward pass: compute ∂L/∂w via chain rule
4. Update: w ← w - α∇L

Optimization Techniques

📊

SGD

Stochastic gradient descent; noisy updates; escapes shallow local minima.

⚡

Momentum

Accumulate gradients; faster convergence; reduces oscillation.

🎯

Adam

Adaptive learning rates per parameter. Works well in practice.

Challenges

  • Vanishing/Exploding Gradients: Deep networks suffer. Solutions: careful initialization, batch norm, skip connections

Convolutional Neural Networks (CNNs)

CNNs exploit spatial structure via local connectivity, weight sharing, pooling. Dominant for image tasks.

Key Operations

• Convolution: Learned filters detect local patterns (edges, textures)
• Pooling: Reduce spatial dims; aggregate (max, average)
• Fully Connected: Final classification layers

Architecture Patterns

  • LeNet: Digit recognition; foundational CNN
  • AlexNet: Large-scale image classification; breakthrough with GPUs (2012)
  • ResNet: Skip connections enable very deep networks (>100 layers)
  • Inception/DenseNet: Multi-scale features; efficient designs

Recurrent Neural Networks (RNNs) and Sequence Modeling

RNNs process sequences via recurrence: h_t = f(x_t, h_{t-1}). Memory of past inputs.

RNN Variants

LSTM (Long Short-Term Memory)
Gating mechanisms: forget, input, output gates. Solves vanishing gradient for long sequences.
GRU (Gated Recurrent Unit)
Simplified LSTM; fewer parameters; comparable performance.

Applications

  • Language Modeling: P(next word | previous words)
  • Machine Translation: Seq2seq: encoder-decoder with attention
  • Speech Recognition: Acoustic modeling with RNNs

Attention and Transformer Architecture

Attention enables models to focus on relevant inputs. Transformers stack attention without recurrence.

Self-Attention

Compute weighted combinations of values based on query-key similarity. Learns what to attend to.

Transformer Components

  • Multi-head attention: Multiple representation subspaces
  • Positional encoding: Inject sequence order (no recurrence)
  • Feed-forward networks: Pointwise MLPs per position
  • Layer norm + residuals: Stabilize deep stacking
Impact

BERT, GPT, T5: transformers dominate NLP. Parallelizable; better for long sequences than RNNs.

Generative Models and Adversarial Training

Generative models learn p(x); useful for generation, density estimation, semi-supervised learning.

Generative Approaches

VAE (Variational Autoencoder)
Probabilistic encoder-decoder. Latent code; reconstruct via gradient-based inference.
GAN (Generative Adversarial Network)
Generator vs. Discriminator game. Generator learns to fool discriminator.
Diffusion Models
Learn to reverse noise addition. State-of-art image generation; stable training.

Training Dynamics

  • GANs: Minimax game; tricky balance; mode collapse issues
  • VAEs: ELBO optimization; stable; sometimes blurry outputs

Regularization and Generalization

Regularization prevents overfitting; generalization good test performance.

Techniques

💧

Dropout

Randomly deactivate neurons during training. Ensemble-like; prevents co-adaptation.

📦

Batch Norm

Normalize activations. Stabilizes training; allows higher learning rates.

📈

Data Augmentation

Transform inputs (rotation, crop, etc.). Increases effective dataset size.

⚖️

L1/L2 Regularization

Penalize large weights. Controls capacity.

Summary: Probabilistic Models & Deep Learning

Probabilistic Models

• Bayesian networks: structure + parameter learning
• EM algorithm for incomplete data
• Graphical models capture dependencies

Deep Learning

• MLPs: universal approximators
• CNNs: image processing
• RNNs/Transformers: sequences
• Generative: VAEs, GANs, diffusion
