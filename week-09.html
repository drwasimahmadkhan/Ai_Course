Communication in Multiagent Systems

Communication enables agents to coordinate, share information, and negotiate joint decisions.

Communication Protocol

Structured message exchange following agreed-upon formats. Enables agents to convey intentions, constraints, and knowledge.

Types of Communication

Broadcast
One agent sends to all. Simple; scalable to many receivers.
Peer-to-Peer
Direct messages between pairs. More targeted; higher latency for group coordination.
Publish-Subscribe
Agents subscribe to topics; publishers deliver relevant information. Decoupled; flexible.
🔤

Speech Acts

Inform, request, promise, decline—structure communication semantics.

🔄

Ontologies

Shared vocabulary ensures mutual understanding.

Argumentation and Negotiation

Argumentation agents exchange reasoned justifications; negotiation reaches mutually beneficial agreements.

Negotiation Protocols

Process

Agents make offers; counteroffers refine proposals. Continue until agreement or disagreement threshold reached.

Argument-Based Negotiation

  • Arguments: Assertions with supporting evidence/reasoning
  • Attack: Counterarguments undermine opponent's position
  • Acceptability: Sets of arguments defended against attacks
Mechanism Design

Design protocols incentivizing truthful revelation, efficient outcomes. Example: ascending-bid auctions reveal true valuations.

Distributed Planning and Coordination

Distributed planning coordinates multiple agents' action sequences toward shared goals.

Approaches

Centralized Planning
One agent plans for all; distribute execution. Simpler; single point of failure.
Decentralized Planning
Each agent plans independently; communicate constraints. Robust; coordination challenges.
Hierarchical Planning
Groups coordinate high-level; individuals detail low-level actions. Scalable; encapsulation.

Conflict Resolution

  • Resource sharing: Prevent collisions, deadlocks
  • Goal conflicts: Negotiate priorities; find compromises

Learning in Multiagent Systems

Multiagent learning enables improvement through interaction while reasoning about others' learning.

Key Challenges

Complexity

Environment non-stationary: others' policies change. Convergence harder than single-agent learning.

Learning Approaches

📚

Q-Learning

Learn value of state-action pairs. Decentralized; independent learners.

📊

Policy Gradient

Directly optimize policy. Gradient directions guide learning.

🎯

Opponent Modeling

Model others' strategies. Exploit weaknesses.

Probabilistic Programming Basics

Probabilistic programming specifies models as programs with random variables; inference computes posteriors.

Core Concept

Write generative model as program: random choices, condition on observations, automatic inference.

Languages and Systems

  • Stan: Efficient Hamiltonian MC for continuous variables
  • Pymc3: Python probabilistic modeling; flexible inference
  • Gen: Julia-based; generative function interface
  • Anglican: Church-like language; sequential Monte Carlo

Inference in Probabilistic Programs

Inference marginalizes over random variables, computing P(latent|observed).

Inference Methods

Rejection Sampling
Execute program; keep samples matching observations. Simple; inefficient for rare events.
Importance Sampling
Weight executions by likelihood of observations. Reduced variance vs. rejection.
MCMC
Markov chain explores posterior. Flexible; convergence diagnosis important.
Variational Inference
Approximate posterior with tractable family. Fast; approximation quality crucial.

Latent Variable Models and Generative Modeling

Generative models specify how data arises from latent variables; invert via inference.

Model Examples

Gaussian Mixture

Data from K Gaussian clusters with unknown assignments. Infer cluster memberships, parameters given data.

Latent Variable Models

  • Factor Analysis: Low-dim continuous factors for dimensionality reduction
  • Topic Models: Topic distributions per document; discover themes
  • State Space Models: Continuous latent trajectories; Kalman filtering

Principle Latent variables enable compact, interpretable representations of complex data.
