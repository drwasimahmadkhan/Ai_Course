Reinforcement Learning Fundamentals

RL learns policies via trial-and-error from environment rewards. Agent maximizes cumulative reward.

MDP + Agent Interaction

Agent observes state, takes action, receives reward and next state. Goal: learn optimal policy π*.

Q-Learning

Algorithm
Q(s,a) ← Q(s,a) + α[r + γ max Q(s',a') - Q(s,a)]
Off-policy; learns optimal Q despite exploratory policy.

Policy Methods

  • Policy Gradient: ∇π ∝ E[∇log π(a|s) Q(s,a)]. Direct policy optimization.
  • Actor-Critic: Actor (policy) + Critic (value). More stable training.

Exploration-Exploitation and Applications

Exploration tries new actions; exploitation uses best known. Balance crucial.

Exploration Strategies

ε-Greedy
Exploit with prob 1-ε, random with prob ε. Simple; fixed rate.
UCB (Upper Confidence Bound)
Balance empirical mean + uncertainty. Principled optimism under uncertainty.
Thompson Sampling
Sample from posterior; act optimally under sample. Bayesian approach.

Real-World Applications

  • Game AI: DeepMind AlphaGo (RL + MCTS + deep nets)
  • Robotics: Learn manipulation, locomotion from robot feedback
  • Recommendation: Contextual bandits for personalized suggestions

Natural Language Processing Foundations

NLP enables machines to understand and generate language. Core tasks: parsing, semantics, pragmatics.

NLP Pipeline

Tokenization

Split text into words/subwords.

Parsing

Syntactic structure; dependency/constituent trees.

Semantics

Word meanings, relations, inference.

Pragmatics

Discourse, coreference, intention.

Word Representations

One-Hot
Sparse vectors; orthogonal; no semantic info.
Distributed (Word2Vec, GloVe)
Dense vectors; semantic similarity ≈ cosine distance.

Language Models and Generation

Language models P(word | context) fundamental for generation, translation, QA.

Transformer Language Models

BERT (bidirectional), GPT (autoregressive), T5 (encoder-decoder). Pre-trained on massive corpora; fine-tuned on tasks.

Applications

  • Machine Translation: Source → intermediate repr → target language
  • Summarization: Compress text preserving key info
  • Question Answering: Extract or generate answers from context
  • Text Generation: Continued completion, paraphrasing, creative writing
Key Insight

Pre-training on unlabeled data → transfer to many downstream tasks.

Semantic Understanding and Knowledge

Semantics map words/phrases to meanings. Knowledge graphs encode structured facts.

Approaches

Compositional Semantics
Build meaning from parts. λ-calculus, formal logic.
Distributional
"Words defined by their use." Similar contexts → similar meanings.
Knowledge Graphs
Entities + relations. DBpedia, Wikidata, YAGO. Enable reasoning.

Inference Tasks

  • Named Entity Recognition: Identify persons, places, organizations
  • Relation Extraction: Discover facts from text
  • Coreference: Link pronouns/mentions to entities

Pragmatics and Dialogue

Pragmatics context-dependent meaning. Dialogue systems maintain state and turn-taking.

Dialogue Components

Dialog State Tracking
Extract user goals, constraints from utterances.
Policy
Decide next system action given state.
Generation
Natural language response from action.

Dialogue Systems

  • Task-oriented: Achieve goals (restaurant booking, tech support)
  • Open-domain: Conversation; empathy, knowledge, coherence
  • Hybrid: Combine retrieval + generation

Information Extraction and Question Answering

Information Extraction pulls structured facts from text. QA answers natural language questions.

Extraction Techniques

  • Rule-based: Pattern matching; precise; low coverage
  • Supervised: Sequence tagging (CRF, BiLSTM); requires labeled data
  • Distant supervision: Weak labels from KB; automatic scale

Question Answering

Types

• Span selection: highlight answer in passage
• Generative: generate answer text
• Open-domain: search corpus + read
